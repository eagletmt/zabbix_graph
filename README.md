# ZabbixGraph

Select hosts and items with peco, and open adhoc graph.

## Installation

    $ gem install zabbix_graph

## Usage

```
$ export ZABBIX_URL=https://your-zabbix.example.com
$ export ZABBIX_USER=...
$ export ZABBIX_PASSWORD=...
$ zabbix_graph
```
